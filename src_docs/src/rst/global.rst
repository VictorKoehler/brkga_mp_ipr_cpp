Overview
~~~~~~~~

All BRKGA-MP-IPR algorithms and facilities are within
namespace ``:ref:`BRKGA<doxid-namespace_b_r_k_g_a>```. Some helper are in
the general scope, such general streaming operator and enumaratio manipulation.
